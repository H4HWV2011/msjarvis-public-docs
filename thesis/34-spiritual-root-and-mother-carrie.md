The `gbim_entities` table in `msjarvis-pgdata-rescue` carries `entity_name`, `entity_type`, and `confidence_decay` fields — the original relational GBIM schema. This table is empty in the current rebuild. It has been superseded by the ChromaDB vector corpus. It is documented as present and empty. No code should write to it in the rebuild.
