# API Specs

This folder contains the public-facing API specification for Ms. Allis.

| Document | What It Contains |
|---|---|
| [openapi.json.md](./openapi.json.md) | The OpenAPI specification for Ms. Allis's public endpoints — documented in Markdown format for human readability, with the machine-readable JSON specification embedded. Covers available endpoints, request/response schemas, authentication requirements, and integration guidance for external partners and developers. |

For a plain-language overview of the API before diving into the specification, see [`docs/API-OVERVIEW.md`](../docs/API-OVERVIEW.md).

For security context governing API access, see [`security/SOURCE_ACCESS.md`](../security/SOURCE_ACCESS.md) and [`security/SECURITY-OVERVIEW.md`](../security/SECURITY-OVERVIEW.md).
