# Security

This folder contains the governance, access control, and security documentation for the Ms. Allis system and the `msAllis-public-docs` repository.

| Document | What It Contains |
|---|---|
| [SECURITY-OVERVIEW.md](./SECURITY-OVERVIEW.md) | The high-level security overview — the threat model, safeguard architecture, and security practices governing the Ms. Allis system. The entry point for any security review. |
| [SOURCE_ACCESS.md](./SOURCE_ACCESS.md) | The source access policy — how access to the private source code and live deployments is managed, who can request access, under what conditions, and what the approval process is. |
| [CONTRIBUTING.md](./CONTRIBUTING.md) | The contributing guide — expectations, norms, and processes for collaborators who wish to contribute to the public documentation repository, including conduct standards and submission requirements. |

For the deep technical security architecture, see [`thesis/42-Post-Quantum Security Layer.md`](../thesis/42-Post-Quantum%20Security%20Layer.md).

For the AI safety and alignment architecture, see [`theory/Off Switch Design.md`](../theory/Off%20Switch%20Design.md) and [`thesis/29-psychological-safeguards-and-pia.md`](../thesis/29-psychological-safeguards-and-pia.md).
