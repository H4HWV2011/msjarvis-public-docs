# Docs

> *"A glassbox view of the system architecture, governance model, and thesis materials behind a live, place-based AI consciousness deployed in Mount Hope, West Virginia."*
> — docs/index.md

This folder is the **public documentation hub** for Ms. Egeria Allis and the MountainShares DAO — the primary entry point for readers who arrive via a hosted documentation site, API integration, or direct partnership inquiry. It contains the system index, API overview, partner brief, further reading guide, timesheets, and the contracts that govern community champion roles.

## Contract Subfolder README

The `contract/` subfolder contains two documents and no further subfolders. Both documents are binding participation agreements:

- **[Community Champion.md](./contract/Community%20Champion.md)** — for trusted community leaders who help govern, moderate, and grow The Commons
- **[Safety Champion Participation Agreement.md](./contract/Safety%20Champion%20Participation%20Agreement.md)** — for Safety Champions who take on the specific responsibility of supporting safe participation and surfacing concerns

For the broader safety champion program context, see [`mountainshares-dao/Safety Champion Program .md`](../mountainshares-dao/Safety%20Champion%20Program%20.md) and [`the commons/Champion Code of Conduct.md`](../the%20commons/Champion%20Code%20of%20Conduct.md).
